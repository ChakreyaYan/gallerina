A lightning fast exhibition design tool for Gallerists, Curators, and Artists!

## How to Run

1.  Clone the repo
2.  Change to the cloned directory from your terminal
3.  Run 'npm install'
4.  Run 'npm start'.  The app will run on http://localhost:3000/ by default
5.  Design some shows!
